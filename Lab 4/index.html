¿Qué diferencias y semejanzas hay entre Java y Javascript?

    Java es un lenguaje de programación orientado a objetos. Javascript es un lenguaje de secuencias de comandos orientado a objetos.
    Java es utilizado para aplicaciones de servidor, escritorio y dispositivos móviles. Javascript es usado para aplicaciones web y front-end.
    JavaScript es interpretado. Java es interpretado y compilado.
    Los dos tienen una sintaxis similar y comparten características básicas como la orientación a objetos y la capacidad de manipular estructuras de datos

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    getDate(): regresa el día del mes
    getMonth(): regresa el mes
    getFullYear(): regresa el año completo (en 4 dígitos)
    getHours(): regresa la hora
    getMinutes(): regresa los minutos

¿Qué métodos tienen los arreglos? (En JavaScript) (Menciona al menos 5*)

    push(): agrega uno o más elementos al final del arreglo
    pop(): elimina el último elemento del arreglo y lo devuelve
    slice(): devuelve una copia superficial de una porción del arreglo, especificada por un índice inicial y un índice final
    join(): devuelve una cadena que contiene todos los elementos del arreglo concatenados entre sí, separados por un separador especificado.
    shift(): elimina el primer elemento del arreglo

¿Cómo se declara una variable con alcance local dentro de una función?

Se utiliza algunas de estas palabras reservadas en las primeras líneas de la función:

    var
    let
    const
Estos van seguidos del nombre de la variable o constante junto a su valor correspondiente

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Pueden causar conflictos con otras variables con el mismo nombre
    El valor de una variable con el mismo nombre puede cambiar accidentalmente
